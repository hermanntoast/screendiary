ScreenDiary

Automated screen recording, OCR, and activity tracking for your Linux desktop. Every moment captured, indexed, and searchable.

Developed and tested on Arch Linux (CachyOS) with KDE Plasma 6 on Wayland.

Quick Start

Get up and running in under a minute.

# Clone the repository
git clone https://github.com/hermanntoast/screendiary.git
cd screendiary

# Install Python dependencies
uv sync

# Copy and edit config
cp config.toml.example config.toml

# Build the frontend
uv run screendiary build

# Install systemd services
uv run screendiary install

# Start everything
uv run screendiary start
